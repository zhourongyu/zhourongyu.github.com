WordPress的半角全角字符转义

發表於 2013-07-17   |   分類於 tech   |  

在WordPress码字中出现了符号中横线”—“ ，页面显示效果却是”—“ 。当你问别人为什么我输入了杠杠却显示了一个大杠呢，相信连专业人士都要思考一会才能明白你的意思。

这个问题学名是：半角全角字符转义，可以简称代码转义。困扰了我很久，今天终于解决了。

WordPress 会智能地将源代码中的所有半角符号自动修正为全角符号，以防止外部源代码在网页上执行。这个功能所带来的麻烦就是在输入代码后，显示的不是原始代码，所以有时候网上复制别人代码会发现不好用。

这个问题应该遇上的不算多，基本只有挨踢的会在博客里写写代码，—这种更多会出现在运维人的博客中～ 既然知道了问题，解决起来就方便多了，方法有多种，这里放出我使用的，并成功的方案（如果你在本文里看到的—是两个的话，说明正确了） 在主题的functions.php 最后添加以下代码：

//取消内容转义 
remove_filter('the_content', 'wptexturize');
//取消摘要转义 
remove_filter('the_excerpt', 'wptexturize');
//取消评论转义 
remove_filter('comment_text', 'wptexturize');

保存后问题应该解决了，WordPress的官方介绍请点击wptexturize

雨后的故宫

發表於 2013-07-04   |   分類於 photo   |  

北京经过几天的雨水，这礼拜的天空异常的好，所以我这两天都背着相机，想找时间去拍几张片子。

昨天下班太晚出门都要天黑了，没得逞。今天下午突然的大雨差点让我以为又泡汤了，幸好雨过天晴，竟然太阳也出来了，下班已经七点了，远地方是去不了了，想了一下决定去景山，因为雨后的天空必然会很美，但要找制高点也不是很好找，市内也就景山还行了。

到景山已经快八点了，景山顶上各种长枪短炮，好不热闹…… 差点连位置都没有，可惜人懒了点没背脚架。最后只能找缝隙拍了几张…

有几点遗憾:

  • 没背脚架，天色以暗，为了保证清晰度只能使用高iso，导致噪点还是不少。

  • 广角不太够，这火烧云的全貌没有得以展现，效果差不少

  • 去的时间还是晚了些，听闻前不久还有彩虹…

最后只能换几个位置拍了几张，拼了幅全景（点击看大图）……

昨天没得逞，只好在路上把看了无数遍的坟包给拍了！！！

行之天津

發表於 2013-06-18   |   分類於 photo   |  

天津号称北方第二大城市，作为通商口岸，被西方多个国家设立过租界，所以很多天津建筑都是国际范儿的～ 和青岛类似，例如意式风情街。

天津小吃也是很出名的，著名的是tian天津三绝：狗不理，十八街麻花，耳朵眼炸糕。切记千万不要因为吃来天津……三样我都吃过，除了麻花还行，狗不理是公认的难吃，炸糕也一般。天津并不算好的旅游地区，可玩之处较少，大部分都是人文建筑，此行也是过来休闲摄影的。

一： 天津之眼。 英国有伦敦之眼，天津也出一个天津之眼。天津真算个山寨之城，不光仿照国外，连国内也不放过。“外滩”正在建设中。放眼过去摊对面高楼大厦还在建设ing……

二、天津市东临渤海，内有海河，算半个沿海城市吧，虽然大部分都是港口，只开放了部分游玩区域，在天津大部分旅行团都会带你们去渤海边游玩，其实那边啥也没有，只能看看海，玩一玩海上项目（坐船）之类的。

还有一处官方名叫东疆湾景区，其实是一个在众多港口中人工填补出一小片海域而建成的。问天津人很多人都不知道，其实是他们都称之为“人工岛”，好土的名字。我们最终选择去了人工岛，从塘沽到人工岛有近30km，打车需要60元左右，门票50/人。 本以为没多少人去，因为去渤海都是10元～，结果到那一看车都停的满满当当，很多北京牌照，大部分都是自驾过来的。 事实证明我们选择没有错， 海滩还是可以玩一玩的～



三、人文建筑。 意式风情街，瓷房子，五大道等。两老爷们看着好别扭….囧…

Clear简介

發表於 2013-06-07   |   分類於 tech   |  


Clear刚出世的时候以令人上耳目一新的设计和交互方式迅速爬到了付费排行榜第一，传说只有了不到10小时。

但发现不少人买了后评分并不高，许多人把它当作todo类的App，和其他gtd软件相比功能差很多，所以不是非常满意。我只觉得它是个备忘录性质的App，所以还是非常满意的。

Clear主要分三层， 菜单、分类、详细事项

主要操作如下：

切换层： 大幅下拉， 详细事项—(下拉)—>分类—(下拉)—>菜单
新建事项： 点击空白处/小幅下拉/双指在两条事项上做张开操作
完成事项： 在单条事项上右滑使该项目变成完成状态，重新打开只需重复操作(右滑)
删除： 左滑
打开收起： 双指进行捏/张开的操作可以缩放当前列表
菜单里有主题、提醒方式等设置，其中一向设置是否在icon上显示数目提醒，这个对有强迫症的人来说是很实用的~

ps：貌似有几个隐藏主题可以开启哦，攻略网上有。

Array与List的那些事

發表於 2013-06-06   |   分類於 tech   |  

Java Api中提供了数组对List的转换方法

String[] tmp = {"a","b","c","d"};

List<String> list = Arrays.asList(tmp);

list.clear() //抛出java.lang.UnsupportedOperationException异常

Arrays的asList实现方式是使用参数引用的数组的大小来new的一个ArrayList，对返回列表的更改会“直写”到数组，而数组没有clear,add,remove等方法。

可以使用CollectionUtils(org.apache.commons.collections.CollectionUtils)进行深度转换

List list= new ArrayList(); CollectionUtils.addAll(list, tmp);

随后对List进行去重操作，方法有多种，这里使用HashSet来实现

public List<String> removeDuplicate(List<String> list) {
    HashSet<String> hs = new HashSet<String>(list);
    list.clear();
    list.addAll(hs);
    return list;
}
