步骤很简单，主要分2步

1，抓取整个HTML页面代码

需要用到了urllib 包，分urllib 与 urllib2 两种，区别可以上自己看。总体来说urllib2比urllib要高一个层次（可以接受Request类的实例来设置URL请求的headers）,HTTP访问大部分都会用urllib2。

ps：urllib 与 urllib2 是不能互相替代的，经常会同时被使用，urllib提供urlencode方法用来GET查询字符串的产生，而urllib2没有。

最基本的抓取：

import urllib2
content = urllib2.urlopen('http://XXXX').read()

这样就得到了整个页面的html代码

2，解析HTML

重要的就是这一步了，如何从一推HTML代码中获取我们想要的信息。这里又提及到了HTMLParser、SGMLParser。当然还有BeautifulSoup等其他一些好用的解析工具，各种优劣比较网上自己查。

这里使用SGMLParser，好处是比较简单，比较通用。另《Dive into Python》中例子用的是SGMLParser。

上面2段html是我们想要的信息，可以看出每个视频都是用 ul 来表示的。信息都在 li 标签下。

这就很明了了，把我们需要的信息从li标签里抽出来即可， 既然是视频，那么URL、名称、发布时间必然是最重要的。

因为默认是按照时间的倒序来排的，这样连发布时间都可以省去，留下最关心的名称及URL。可以看到这两个信息都可以在

<li class="v_title"><a title="xxx" target="video" href="xxx">xxxx</a></li>

这么一个class 为v_title的

  • 标签里。 href指向的就是我们需要的URL，内是我们需要的名称。

    • 1.进到class=’v_title’的li标签

    2.找到标签的href地址放入urls列表中

    3.提取的text数据放入title列表中

    4.按<名称 URL>循环打印。

    简洁明了，直接上代码吧：

    #HTML解析类GetVideos
import urllib2
from sgmllib import SGMLParser

class GetVideos(SGMLParser):
    def __init__(self):
        SGMLParser.__init__(self)
        self.titles = []
        self.hrefs = []
        self.flag = False    ##<li>标记
        self.data = False    ##<a>标记

    def start_li(self,attrs):
        for k,v in attrs:
            if k == 'class' and v == 'v_title':   ##进到<li class="v_title">
                self.flag = True
                return

    def end_li(self):
        self.flag = False

    def start_a(self,attrs):
        if self.flag == False:
            return
        self.data = True
        for k,v in attrs:    ##进入<a>，并把href标签数据保存
            if k  == 'href' and v:
                self.hrefs.append(v)

    def end_a(self):
        if self.data:
            self.data = False

    def handle_data(self,text):
        if self.data:
            self.titles.append(text)

    def printInfo(self):    ##打印所获得的数据
        for title,item in map(None,self.titles,self.hrefs):
            print title.decode('gbk','ignore').encode('utf8'),item    ##将GBK转UTF8，遇特殊字符则ignore

    iPhone那些你不知道的小技巧

    發表於 2012-04-15   |   分類於 tech   |  

    很多朋友经常的问我关于iPhone的许多技巧，国内确实有很多玩不明白的小白～
    iPhone有很多实用的小技巧，还是值得一学的，如果你真的想做一个iPhone达人的话……

    技巧1：
    截屏功能，个人认为最实用的功能，以前用诺基亚都得下载各种截屏软件，挺麻烦，iphone默认提供了这个功能，在当前界面按HOME+开机键既可把当前屏幕截取下来，默认存在相册中

    技巧2：
    旋转屏幕，有时候躺在床上看书或者网页，屏幕会自动的随着旋转，有时候确实很烦人，这时候需要按HOME键2次，底部会出现一排图标，都是近期使用过的app，滑动东到第一页（手指向右侧滑动），点一下第一个图标，在电池左侧会出现一个小锁，代表了屏幕锁定，不会再随着重力感应旋转了

    技巧3：
    表情键盘，iPhone是可以默认打出各种表情图片的，相信很多人都见过，遇到新的表情甚至会问对方是打哪个词语而出现的表情，其实iPhone是带有一个表情键盘的，大约有450多种表情都在里面，在设置—通用—键盘—国际键盘—添加键盘—表情符号。添加完后在输入界面就可以切换到表情符号键盘（左下角的地球符号）。有人说要安装一个叫emoji的app，运行，然后才能添加，在我的4s下是可以直接添加的，其他的机型没有测试过。

    技巧4：
    编辑短信的时候如果想把所写的内容全部删掉，可以试着晃动几下手机，会弹出个窗口，选择“撤销键入”就可以把内容全部删除，如果删除错了想恢复，再次晃动手机，选择“重做键入”既可。如果是粘贴过来的内容，则会出现“撤销粘贴”。

    技巧5：
    摄氏度，在输入内容的时候如果想输入摄氏度这个符号“ °C ”，切换到数字输入界面，长按0会出现一排类似的符号哦，第一个就是摄氏度的°

    今天彻底查了下屌丝到底什么意思，这个是还算满意的结果。
    屌丝一词源于百度李毅贴吧，是一种带有谐谑语气的称呼，可用于称呼别人，亦可自谓。屌丝二字蕴含着无奈与自嘲的意味。但凡被称作屌丝的，或者自称为屌丝的人都有一些共同特征。
    起源
    表面上屌丝一词起源于百度雷霆吧对李毅的球迷无用心的恶搞称谓。但广大毅丝能接受了这个称呼并发扬光大，深沉次的原因还是因为每个人心中都有屌丝一面。

    特征
    成长历程：屌丝一般都是农村出身，小时家里贫穷。有的十二载寒窗考上大学，攻读工科专业，毕业后工作类似搬砖、码农，毫无地位、收入微薄；有的初中辍学，进 城务工，或成了发廊师傅，或成了网吧网管，在城市的繁华之中分得一杯苦羹；或是无业游民，但是自己一般不愿承认，网上经常以自由职业者。

    中天门-南天门索道：80/单程，往返140（除万不得已，否则尽量不坐）

    登山装备：
    四月的泰山气温还很低，又是夜间登山，没上升1KM海拔气温约降低 6°C - 8°C ，预计夜间山顶气温在-10°C左右（可能更低），建议备选一件羽绒服（推荐着装：贴身内衣+棉制长袖或卫衣+羽绒服），裤子不要太薄，防风最好，虽然在山顶会租借军大衣，下半身还是要穿暖和些，以防万一。
    鞋子为登山中最重要的装备，选择轻便，透气，鞋底摩擦力大的鞋子，严禁高跟鞋，皮鞋，厚底鞋。

    泰安特色餐饮：

    2.小米煎饼和烧烤（天街，李胖子烧烤）

    3.夜间山顶风大，手套、帽子、围脖都是御风防寒的好物品，建议携带
