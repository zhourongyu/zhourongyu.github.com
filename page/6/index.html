OSX Mavericks(10.9) 睡眠后唤醒无声音的解决方案

發表於 2014-01-23   |   分類於 tech   |  

最近时常发生睡眠后唤醒无声音，开始以为是耳机问题~
经尝试发现确实是系统问题，Apple也确认了这个问题，目前共有三套解决方案

  • 坐等系统更新
  • 重启，麻烦的手段
  • 重载驱动

当然方案三最靠谱，重载驱动命令：

sudo kextunload /System/Library/Extensions/AppleHDA.kext
sudo kextload /System/Library/Extensions/AppleHDA.kext

2014 Mac装机手册

發表於 2014-01-14   |   分類於 tech   |  

2014年最新的Mac新机配置单，仅供参考

基础组件：

Homebrew

Homebrew是mac下最好用的包管理器，安装方式：

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

eg: brew install wget

oh-my-zsh

zsh是mac下bash的替代品，或者说是bash的增强版，有不少强大的插件。 oh-my-zsh是zsh常用的配置包。

iTerm2

iTerm是Mac自带终端的增强版

开发工具

Command Line Tools：基本开发命令工具集合
Dash：语法库文件本地查询与生成工具
Github for Mac：Github官方客户端
IntelliJ IDEA：对mac支持最好的JAVA开发工具

通用软件

云端软件：
Evernote：强大个人云笔记，横跨各个平台
Dropbox：文件云端同步利器

输入法：
百度输入法：百度输入法是最近换过来的，经过试用和QQ输入法旗鼓相当
qq输入法：稳定，更新教快

写作与编辑：
Sublime Text 3：编辑器神作
Mou：简洁，实时预览的Markdown写作神器

系统增强：
Alfred 2：必装神器之一，配合work flow可出现神龙
Keka：免费压缩/解压软件
Thunder：比Windows下简洁太多，好太多
MPlayerX：影音播放器

社交：
QQ：社交老大

摄影：
Aperture 3：苹果自家强大相片管理+调整软件，iPhoto增强版。
Lightroom 5：专业摄影照片调整工具

Apple特别购物日

發表於 2014-01-12   |   分類於 life   |  


今天是苹果优惠购物日，每年苹果会在世界各地举办新年优惠活动，不同国家方式不同。比如日本的福袋就挺有意思…大陆的活动历来感觉一般。去年记得ipad什么的减价大约是300左右。今年果然也差不多~让苹果打个折太不容易了…

在今天下手了一台ME294，国行官网订购，最大的好处是招行/工行的分期0手续费，相当的给力。最大支持12期0手续费，如果把这些钱放到余额宝里，按现在的收益率6%来算。 6%*18000 = 1000元左右，也就是说，在优惠700元的前提下，还能省下好几百。

重量2.02kg，比普通的13寸MBP还要稍轻一些，当然还是比Air重一斤多。总体还是非常能接受的。

关于Retina屏幕的效果，记得之前在Apple Store里看，并没有觉得很惊艳。
到手后特意和自己的MBP做了仔细对比，在相同设置下访问同一网站/网页，咋看起来差不太多，但仔细一看会感觉RMBP非常锐，越看越想…可以就像看单反和卡片拍出相片~感觉一切都是那么的漂亮。

经过几天的使用，我确实再也不想看MBP的屏幕了…连自己的Air也丢到一边去了！

这是一台买了绝对不会后悔的神器，2013令人最具幸福感产品之一！
