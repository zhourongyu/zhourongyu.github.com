苹果还是那个苹果

發表於 2013-09-11   |   分類於 life   |  

昨晚本来想熬夜看直播，不料年事已高，卧床至12点左右竟然睡着了…早上起床发现电脑电都放完了。只能看看各大媒体的总结了，让人欣慰的是在没有乔布斯的这段日子，苹果还是那个苹果，也许创新能力下降，但思想犹存。

一，全球第一款使用x64的手机。在我看来最大的亮点是苹果明智的选择了X64，而不是像Android阵营那样都四核八核的往上堆，让人看起来很牛逼的样子。说起64位又让我想起了Windows，不知道现在有多少朋友还在害怕64位的系统，在Windows刚出现x64的时候问题太多了，各种系统问题，软件兼容问题。为了兼容C盘下一直存在(x86)和(x64)两个目录，这真让人恶心。当苹果的OSX要转x64的时候不少人也担心像Windows那样出现问题，但是直到现在我一点感觉都没有，甚至认为这转了不和没转一样吗？装软件用软件从来没变化……真是人比人气死人。

今天一台双核电脑给大部分人用都可以完全胜任，一个手机有必要牛逼到八核的概念吗，还是说这个八核完全是炒作（确实是给人装逼用的）。硬件的升级涉又及到耗电，内存等各方面的优化，耗时费力，不是一味的堆配置就是牛逼。

二，Touch ID （指纹识别），看起来不像Siri那样鸡肋，这个名称取的很大，应该不单单只是个指纹识别。可以通过指纹验证购买APP有点意思，我认为未来指纹支付的模式更优于NFC。现在NFC大热，各大商家都在玩NFC的时候，Apple依旧有自己的思想（好像一直都不看好NFC）。至于实际使用还有待实测，指纹技术应该够成熟了，只是Home区域很小，只能覆盖部分指纹，加上单手操作，识别率有待考察。这个看样子又会引起一堆争议，just like siri……

三，5c颜色很亮，合适青春活力资金不太宽裕的学生/小青年，切记不要买国行，一定要港行~
5s值得吐槽的也是颜色。 白依旧靓丽，所谓的空间灰不知道什么概念（貌似是比5要浅），希望比5的要好。金色嘛，做好了是高贵色，做不好就是暴发户手机。我喜爱的高贵黑色再也找不到了…

另外一件有意思的事是：Tim Cook的几次访华没白来，中国第一全球同步发售iPhone，香港都不在第一批国家内，香港会不会过来抢大陆”水”货（估计够呛）。今天国内的发布会要出TD-LTE的5s了。这个才是我最想看到的！

关于屏幕一直都徘徊，iPhone 5的4寸屏幕已经是我的操作极限（右手单卧拇指点击左上角按钮）。大屏始终有更好的视觉体验，但也带来了更大的消耗，从体积，视觉，操作来说对目前的屏幕还是满意的，但还是希望Apple出个大屏的，毕竟还是有不小的市场。一直认为一个男人拿着五六寸手机，还是那种超薄的感觉非常”娘”（例如Note系列），裤兜还不好塞，塞包里更娘。五寸已经够勉强了，使用Sony那非人类的6.5寸手机绝非普通人。。。

骑车的路上（一）

發表於 2013-09-03   |   分類於 life   |  

骑车上下班已经一月有余，每天来回路程45km左右，看着不太远，但总路程算下拉竟然也有1000km+了，还是有点长的，从北京到我家的距离大概1500km。 记得有一天给妈妈打电话说：妈，或许哪天我头脑一热骑车回去了。把我妈吓一跳，让我别犯傻～

骑车的路上也有很多有趣的事情，所以我打算把它们都记录下来。我每天早上出门的路线都是抄小路出发，因为大路堵车比较严重，虽然我很喜欢以S形路线穿梭在拥堵的车流中，但还是影响了我的速度。 而晚上回家的路线是一直走大路，因为车少还有路灯。 今天因为懒的等红灯抄小道回家了，但我惊奇的发现我走了无数遍的小路竟然不认识了，一度让我怀疑是不是走错了，最终怀着一路的疑问还是正确的到达了终点。

回来后我又思考了下这个问题，是因为逆行的原因吗？是靠右骑车进而忽略了左边的“风景”吗？但就连很小的一个车道也像是完全陌生的感觉。 这就是角度带来的变化，就像倒立看世界一样，当你对倍感无聊，对两点一线，每日如一的生活失去兴趣的时候不妨尝试一下变换自己的角度，也许又是一番精彩的世界。

其实我很早就发现并运用过这种方法。下面是两张在云蒙山拍的照片，右图是我拿出来给朋友们看的，仔细看，有没有感觉到视觉上的别扭，无法猜测出是什么角度拍出的照片。不光是角度，连彩旗都往上漂。左边的就比较正常，是在一排彩旗下面仰拍的蓝天。

右图只不过是把仰拍的图片倒置过来的而已。因为不同的视角才能有新鲜的感觉。摄影的魅力也就在于让你发现更多不同的视角。

Sublime Text 3 公测

發表於 2013-08-26   |   分類於 tech   |  

Sublime Text 3(下载地址)已经公测了。 Python版本已经更新到3x，导致很多插件不好用,。
Package Control安装方式也无法使用，官方最近也给出了最新的安装方式：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

另外也可以单独下载Package Control.sublime-package 放入Installed Packages/中，然后手动选择安装。

Sublime Text 3的主要改进：

  1. 强大的加载速度，基本等于秒开

2.新增Go Definition , Sublime Text 2最大的缺憾就是不能跳转到方法，只能借助ctags

3.新增错字的模糊匹配，例如main 输入min也有提

WordPress的半角全角字符转义

發表於 2013-07-17   |   分類於 tech   |  

在WordPress码字中出现了符号中横线”—“ ，页面显示效果却是”—“ 。当你问别人为什么我输入了杠杠却显示了一个大杠呢，相信连专业人士都要思考一会才能明白你的意思。

这个问题学名是：半角全角字符转义，可以简称代码转义。困扰了我很久，今天终于解决了。

WordPress 会智能地将源代码中的所有半角符号自动修正为全角符号，以防止外部源代码在网页上执行。这个功能所带来的麻烦就是在输入代码后，显示的不是原始代码，所以有时候网上复制别人代码会发现不好用。

这个问题应该遇上的不算多，基本只有挨踢的会在博客里写写代码，—这种更多会出现在运维人的博客中～ 既然知道了问题，解决起来就方便多了，方法有多种，这里放出我使用的，并成功的方案（如果你在本文里看到的—是两个的话，说明正确了） 在主题的functions.php 最后添加以下代码：

//取消内容转义 
remove_filter('the_content', 'wptexturize');
//取消摘要转义 
remove_filter('the_excerpt', 'wptexturize');
//取消评论转义 
remove_filter('comment_text', 'wptexturize');

保存后问题应该解决了，WordPress的官方介绍请点击wptexturize

雨后的故宫

發表於 2013-07-04   |   分類於 photo   |  

北京经过几天的雨水，这礼拜的天空异常的好，所以我这两天都背着相机，想找时间去拍几张片子。

昨天下班太晚出门都要天黑了，没得逞。今天下午突然的大雨差点让我以为又泡汤了，幸好雨过天晴，竟然太阳也出来了，下班已经七点了，远地方是去不了了，想了一下决定去景山，因为雨后的天空必然会很美，但要找制高点也不是很好找，市内也就景山还行了。

到景山已经快八点了，景山顶上各种长枪短炮，好不热闹…… 差点连位置都没有，可惜人懒了点没背脚架。最后只能找缝隙拍了几张…

有几点遗憾:

  • 没背脚架，天色以暗，为了保证清晰度只能使用高iso，导致噪点还是不少。

  • 广角不太够，这火烧云的全貌没有得以展现，效果差不少

  • 去的时间还是晚了些，听闻前不久还有彩虹…

最后只能换几个位置拍了几张，拼了幅全景（点击看大图）……

昨天没得逞，只好在路上把看了无数遍的坟包给拍了！！！
