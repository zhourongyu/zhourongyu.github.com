西安西安

發表於 2013-01-14   |   分類於 photo   |  

夜已深，不想在碰电脑，试着用ipad来完成。
从西安回来十多天了，选择在这个不合时宜的时间去西安也是随意之旅，恰好有便宜机票。
中学历史学的并不好，西安给我的印象除了古都就剩下兵马俑，当然还有西岳华山。
这次西安的行程很随意，完全随心情。
说不合时宜是有依据的，元旦温度很低，零下十度左右，导致很多地方也没开发，例如华山著名的鹞子翻身与长空栈道。也是最遗憾的地方！好在顶着寒风硬是看完了日出。体会了僵硬的状态。

站在城墙上还是有古都的感觉，加上冬天天气一般，人烟稀少。在寒风中漫步在城墙上有种风萧萧兮易水寒，壮士一去兮不复返的感觉〜偶尔身边会冲出几辆自行车，有赶时间回去还车的〜超时要加钱了〜


站在城墙上我想起电视剧中那些攻城用的云梯，在现场看来得非常高，光爬这么高且陡得梯子也非一般人敢为。何况搬运的人有多苦。


到哪都有潮流，看我们的城墙Style！模特都是好基友Cao来担任。

城墙很长13km左右。步行可以费很长时间，我们就等到了夕阳，只可惜现在城墙周围楼高房大，不然可以看到很美的落日景象

城墙的夜景也相当漂亮，可惜再最后走的时候才发现，大包小包的没法拍了。
不论是兵马俑还是博物馆都是去学知识的，如果历史课都是这样现场来讲，相信所有人都能学的倍儿棒。

来西安不吃小吃是不可能的，不过现在全国小吃一条街都类似，什么都有。既然在西安那就多吃羊肉泡馍，BiangBIang面，肉夹馍等类似食物，在我看来不太符合我的口味习惯，因为每天都在愁吃什么〜我还是更偏向北方的面食。
重点提下羊肉泡馍，真的要自己撕，不熟练的我们撕了二十分钟有，手指都疼了一天。。。吃个泡馍不容易

关于华山也要说几句，华山在大家眼里是险字，实际上只是坡度稍大一些，对于其他山来说确实较险，不过也没有危险。真正的险也许是那条古老的真正的智取华山路线，景点内开放的那条是假的〜
华山的观景台在东峰，也许是因为高度不够，位置并不是太好，有山峦挡住了地平线，不不算最好的看日出位置，相比泰山、峨眉还是有点差距的。不过也是另一种感觉。

最后可以看看冻僵硬后还没完全恢复的我。此时已经接近中午了，地处南峰，华山之巅。身后这块石头上面应该就是华山的最高点了。
一个身穿一件T恤、一件卫衣加一件普通防风帽衫在夜爬之后扛着脚架，背着相机顶着零下十多度寒风等日出的不要命的孩子

好了，用pad打这么多字，手都酸了。下次还是电脑吧…………

Ipad批量删除相片

發表於 2013-01-09   |   分類於 tech   |  

用过Camera Kit的朋友应该深有体会…
PS:Camera Kit是应用于ipad1-ipad3上的，ipad4接口换成了Lightning to SD Card Reader。两件拆开卖了～更贵，而且没有原来方便（个人认为）

每次旅游回来回有很多照片，导入ipad后至少都有上百张。但是导入后想再删除这些那就累死了，因为只能一张一张删，最近才知道原来ipad有多选的操作。。
在Photo （照片）App中进入待删除的相簿（不是进入单张照片，而是进入到缩略图模式）——点右上角的Edit（编辑）—— 用两只手指轻点并停留在要删除的第一张照片上，继续保持双指，扫过需要删除的照片。 这个时候你可以看到手指扫过的照片已经被选定。重复操作直到需要删除的照片被全部选定，点删除即可。

Mac OS X系统下刷新DNS缓存的方法

發表於 2012-12-14   |   分類於 tech   |  

在Lion(10.7)之前，刷新DNS缓存的方法是：

互联网上搜索 Mac OS X 系统下刷新 DNS 缓存的方法，绝大部分网站都说是在终端中执行下面这个命令：

sudo dscacheutil -flushcache

但其实上面的命令仅针对 Mac OS X 10.5 和 10.6 有效，在 Lion 和 Mountain Lion 中都是错误的，正确的应该是：

sudo killall -HUP mDNSResponder
