HTTP的状态码

發表於 2011-12-07   |   分類於 tech   |  

ps:昨晚服务器出现严重错误，出现504，老大误以为是新上的负载均衡引起的故障，后经排查是因为统计库日志被写满～

下面是一些常见的状态码：

200 OK 一切正常
301 Moved Permanently 永久性的转移
302 Found 暂时性的转移
400 Bad Request 请求出现语法错误
403 Forbidden 资源不可用
404 Not Found 无法找到指定的资源
500 Internal Server Error 服务器遇到了不可预料的情况
501 Not Implemented 服务器不支持实现请求的功能
504 Gateway Timeout 不能及时的从服务器获得应答<
/pre>
500是大家最不想看见的错误～ 因为错误的可能性很广，不易排查。

说说301 和 302，近日在研究nginx，很多人都说nginx目前不支持.htaccess文件，不支持rewrite。

其实它是支持的，依赖pcre库。

301和302的区别主要在于一个是永久跳转，一个是临时跳转。

为了页面的SEO，应尽量使用301的方式，如果我们把一个地址采用301跳转方式的的话搜索引擎会把老地址的PageRank等信息带到新地址，同时在搜索引擎库中彻底废掉原来的老地址。

例如在本站中，需要把zhourongyu.me, zhourongyu.me等域名自动跳转到www上。

server{
    listen 80;
    server_name *.zhourongyu.me;
    access_log off;
    rewrite ^/(.*)$ http://zhourongyu.me/$1 permanent;
}

那302的跳转一般用在哪呢？
在一些需要临时的功能上的跳转的时候，例如JS的跳转，http的meta跳转会发送302状态码。
另外302一般也会作用与恶意的URL劫持（URL hijacking），～感兴趣可以去研究下！

CentOS安装Python

發表於 2011-12-05   |   分類於 tech   |  

centos5.6自带的是python2.4，版本太老。

如果需要使用高版本的python 则需要自己 编译并安装

编译python 前需要先安装 编译环境

使用 yum install gcc gcc-c++ autoconf automake 来安装编译环境

使用wget下载python

wget http://www.python.org/ftp/python/2.7.2/Python-2.7.2.tgz

tar xzf Python-2.7.2.tgz
cd Python-2.7.2

编译安装python

./configure --prefix=/usr/local/python2.7
make
make install

创建一个python2.7的链接

ln -sf /usr/local/python2.7/bin/python2.7 /usr/bin/python（可以取名python2.7，这样方便安装多个版本）

and 直接运行OK

帝都工作日乘地铁秘籍

發表於 2011-10-31   |   分類於 life   |  

高效进站
进站出的限流S形通道一定要走外侧（大部分人认为内侧更近，都堵在内侧，学过数学的都知道是一样的~ 走外侧能比内侧最少快1/3时间 ），途中如有入口分支要及时切换到到邻近内道，渡过拥挤处再切回外道。

车厢选择
大部分人都懒或者是想快速出站，选择离入口及出口最近的几个车厢，为了避免人数过多，因选择车头及车尾附近，一是人较少二车头和车尾的等车处位置很小，不能容纳很多人，队形很可能是是4x3或者5x3，而普通的队形是2xN. 其次因选择中部车厢，此处里入口出口较远，人数也相对较少。

安全上车
地铁门约三人宽一般站位是中间2人左右个半身，共四人，上地铁时要谨记宁可站中间三四排也不站左右侧二排（在开门瞬间强大的冲击力会将左右外侧一排以后的人全部挤在门外，中间反而落座几率更大）

落座选择
上车后切记不要左右徘徊，上车前应该算好 左右方的人数，拿中间车门上车来说，一排座位6人，左右共4排，平均分配的话是3x4=12个座位，首先通过言语，眼神预判身边人的预期走位，选择一个人少的方向，定好目标迅速落座，如不幸上车后选好的坐被人稍快零点几秒捷足先登，可以顺势挤在他/她身旁，通过左右的冲击有一定的几率挤出一个位置……

站位选择
如果不幸没座或者是中途上车的人一定不要站过道上及门旁边扶手处，因选择坐席面前的通道（最拥挤的地方莫过于进门初的通道，这点坐多了的人都知道，但有人喜欢躲在门旁边的扶手处，其实没有好效果，一边被人挤，一边还会妨碍座位上的人，被人暗骂），在坐席的过道处可以找一个你喜欢的Man or Girl 依靠一下，既舒适又开心。

目标选择
要在上车后选择好目标，上车后凡见到呼呼大睡者必定不会短时间下车，首选座位上四处张望者，其次经常看一眼手机或者手不停的拎着在旁边的包包上的人。注意：中途发现更适合目标一定要及时换位。

卡位法
选择好目标后记住不要站在目标的正前方，要站在目标与左或右放的人中间，这样几率更大，体型瘦弱者要张开双臂，双手抓上横杆，增加自己宽度，一旦目标起身，方便落座。
