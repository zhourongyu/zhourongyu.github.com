Diskpart做U启

發表於 2011-06-09   |   分類於 tech   |  

还记得为了能够迅速稳定安装XP的时代么，做PE，GHOST等，当时还乐在其中，非常有成就感，知道NT 6.0以后，似乎一切变得简单了，因为有DISKPART。

从Windows NT开始，Windows已经彻底从曾经的MS-DOS中分离出来了，但是直到现在，仍然有很多人在使用MS-DOS下的FDISKS工具来对磁盘进行分区，接下来，我就介绍一下Windows NT的磁盘分区工具DISKPART。

DISKPART 可以做的事情太多了，下面只是对一个U盘做了U启方便了日后为自己或他人安装WIN　NT6.0以上都系统（vista，win７）。

在命令提示符下输入diskpart即可启动，注意：你必须拥有管理员权限。
接下来的例子中，我对一个新的U盘做U启，将其格式化成NTFS格式。

list disk
select disk 1　（注意看disk大小，千万不要选中自己电脑的硬盘，一般从０开始）
create partition primary
select partition 1
format fs=ntfs quick
active
exit

够简单了吧，不必再去网上找各种制作U盘启动都工具了！
